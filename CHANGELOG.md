# Changelog

All notable changes to this repository will be documented in this file.

The format is inspired by Keep a Changelog, and this project currently uses a simple manual versioning approach while the MVP is still moving quickly.

## [0.1.0] - 2026-04-19

### Added

- Initial GitHub-ready FORE MVP repository scaffold
- Next.js App Router structure for homepage, markets, market detail, portfolio, rewards, rules, help, and create pages
- Shared demo account and wallet state using React context
- Market discovery with search, category filtering, and status filtering
- Trade panel with buy flow, quick amount shortcuts, payout preview, and position-aware actions
- Portfolio interactions for close and claim flows
- Mock API endpoints for markets and account session data
- Cold-white product UI with launch-oriented visual system
- Expanded README with architecture, roadmap, FAQ, and operational project information
